# T1 — 选择根目录 + 递归扫描含 CLAUDE.md 的文件夹 + 路径前缀去重

## 技术原理

Node.js `fs.readdirSync` 递归遍历目录树，遇到 CLAUDE.md 文件即记录该目录。  
**前缀去重**：对找到的路径按长度升序排序，若某路径是已接受路径的子路径（startsWith(parent + sep)），则跳过——只保留最外层的 CLAUDE.md 目录，避免子模块重复列出。

## 原子目标

- [x] ProjectScanner.ts：`scanForProjects(rootDir)` 递归扫描（depth ≤ 6）
  - 排除目录：`node_modules .git .npm dist build out __pycache__ .venv venv .next .cache`
  - 返回 `ScannedProject[]`：`{ path, name, isGitRepo }`
- [x] `isGitRepo(dir)`：检查 `dir/.git` 是否存在
- [x] `deduplicateByPrefix(paths)`：路径前缀严格去重，保留最浅路径
- [x] IPC `DIALOG_OPEN_DIR`：触发 Electron 原生目录选择对话框
- [x] IPC `PROJECT_SCAN`：接收 rootDir，调用 scanForProjects 返回结果

## 验收标准

调用 `PROJECT_SCAN` 传入 `~`，扫描结果仅包含顶层 CLAUDE.md 项目（嵌套子项目不重复出现），控制台打印扫描路径数量。
