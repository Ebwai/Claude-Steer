# T1 — 3步向导覆盖右侧面板

## 技术原理

`GlobalMonitorPage` 管理 `wizardOpen: boolean` 状态，当 `wizardOpen === true` 时渲染  
`CreateProjectWizard` 替代 `RightPanel`（同区域 CSS 覆盖）。  
"+" FAB 按钮通过 `onCreateProject` 回调传入 `CanvasPanel`，点击触发 `setWizardOpen(true)`。

## 原子目标

- [x] GlobalMonitorPage.tsx：添加 `wizardOpen` state，条件渲染 Wizard 或 RightPanel
- [x] CanvasPanel.tsx：FAB "＋ 新建项目" 接受 `onCreateProject: () => void` prop
- [x] CreateProjectWizard.tsx 三步骨架：StepNav（三圆点步骤指示器）+ 内容区 + 底部按钮
- [x] Step 1 (项目设置)：项目名 + 路径选择（DIALOG_OPEN_DIR）+ 权限模式下拉
- [x] Step 2 (放入资产)：展示路径 + "打开文件夹" + 说明文字（异步操作，可跳过）
- [x] Step 3 (制定计划)：初始 prompt textarea（发送给 Claude 的第一条指令）

## 验收标准

点击"+"按钮，右侧面板切换为3步向导；可在3步之间前进后退；取消按钮恢复 RightPanel。
