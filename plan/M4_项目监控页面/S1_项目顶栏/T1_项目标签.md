# T1 — 浏览器式项目标签

## 技术原理

**浏览器式标签**：仅在顶部标签栏显示 claimStatus=1 且 activeSessionId 非空（运行中）的项目，末端有"所有项目"折叠按钮展开所有已认领项目。标签选中状态保存在 `activeProjectIdAtom` 中，驱动整个 ProjectMonitorPage 的数据渲染。

**架构设计**：
- `ProjectTabBar` 组件：从 `claimedProjectsAtom` 读取，过滤运行中的项目渲染为标签
- `activeProjectIdAtom`：新 Jotai 原子，存储当前选中的项目 ID
- "所有项目"展开：使用浮层下拉（不是路由跳转）

## 实现步骤

### 原子目标 1 — 新增 activeProjectIdAtom [ ]
- 在 `src/renderer/src/atoms/projects.atom.ts` 添加：
  ```typescript
  export const activeProjectIdAtom = atom<string | null>(null)
  ```

### 原子目标 2 — ProjectTabBar 组件 [ ]
文件：`src/renderer/src/features/project-monitor/ProjectTabBar.tsx`
- Props: 无（从 atom 读取）
- 读取 `claimedProjectsAtom`，`activeSessionsAtom`，`activeProjectIdAtom`
- 运行中项目 = claimedProjects.filter(p => p.activeSessionId && session.status === 'Running')
- **每个标签规格（height: 34px）**：
  - 绿色状态点（6px 圆点，running=绿色/interrupted=橙色）
  - 项目名称文字
  - ✕ 关闭按钮（**只关闭该项目的监控 tab，不终止进程**，PTY 继续在后台运行）
  - 当前激活 tab：高亮背景 + 深色底部边框（2px）
- 末端「所有项目 ▾」按钮 → 浮层显示**全部**已认领项目（含未运行），选择后切换到该项目监控页面
- 点击标签设置 `activeProjectIdAtom`

### 原子目标 3 — ProjectTabBar.css [ ]
- `.ptab-bar`：flex row，高度 38px，背景 `var(--bg2)`，底部边框
- `.ptab`：tab 单元，padding 0 14px，flex center，gap 6px，hover/active 状态
- `.ptab-active`：选中态，底部 2px orange 下划线，背景 `var(--bg1)`
- `.ptab-dot`：状态圆点 6px
- `.ptab-all-btn`：末端按钮，右侧分隔线
- `.ptab-dropdown`：绝对定位浮层，`var(--bg2)` 背景，z-index: 200

### 原子目标 4 — 集成到 ProjectMonitorPage [ ]
- App.tsx 中 `activeTab === 'project'` 改渲染 `<ProjectMonitorPage />`
- ProjectMonitorPage 顶部渲染 `<ProjectTabBar />`

## 验收标准
- 有运行中项目时标签出现在顶部
- 点击标签切换选中项目
- "所有项目"浮层展示全部已认领项目
